# catdancer

猫が見て楽しむための全画面 Web アプリです。画面内をオブジェクト（ネズミ・猫じゃらし・おもちゃ・虫など）が動き回り、動きに連動した効果音（SE）で猫の狩猟本能を刺激します。PixiJS v8 + Vite + TypeScript で実装しています。

## 概要

ブラウザを全画面表示にして猫の前に置く用途を想定した、単一ページのアプリです。2 つのモードを備えます。

- **マウス操作モード（`manual`）**: 選んだオブジェクト（既定はネズミ）がマウスカーソル（タッチ位置）に反応します。操作対象はオプション画面で切り替えられます（ネズミ／猫じゃらし／おもちゃ／虫）。
- **猫用動画モード（`auto`）**: オブジェクトが一定間隔で自動的に画面外から現れ、画面内を動き回り、やがて画面外へ退場します。

設定はオプション画面から変更でき、`localStorage` と `IndexedDB` に永続化されるためリロードで復元されます。

## 機能

### モード

- **マウス操作モード**: オプションで選んだ操作対象（既定＝ネズミ）を 1 つだけ画面に出します。操作対象ごとに挙動が異なります（下記「マウス操作モードの操作対象」）。
- **猫用動画モード**: 登録済みの種別を重み付き乱択で選び、画面外（world 端）から出現させます。種別ごとの動きで画面内を動き、world 外へ抜けると自動的に消滅（despawn）します。同時出現数は上限（既定 12 体）で頭打ちします。

### マウス操作モードの操作対象

操作対象はオプション画面の「マウスモード」タブから切り替えます。実行中に切り替えると、前の対象を後始末してから新しい対象へ即座に差し替わります（同時に出るのは常に 1 種類）。

- **ネズミ**（既定）: カーソルへ臨界減衰スムージング（SmoothDamp）で俊敏に追従します。進行方向へ 360 度回頭し（左半分は鏡像反転で上下を自然に保つ）、本物のテクスチャ（`mouse-tail.webp`）を使ったワールド空間の物理トレイル尻尾（Verlet 質点鎖）が後方に流れます。カーソルが画面外へ出るとネズミも画面外へ走り去り、戻ると再び画面内へ現れます。クリック／タップで鳴き声（チュー）を鳴らします。
- **猫じゃらし**: 画面端から大きな穂を差し込み、穂先がカーソルへバネ的なラグで追従します（速い振りで遅れて振れる「ふりふり」）。飛び出す端はカーソル位置に応じて可変で、カーソルが端に近づく／画面外へ出ると穂を端の外へ引っ込めます。音・尻尾はありません。
- **おもちゃ**: カーソルへ追従します（ネズミと同じ追従挙動。尻尾・音はなし）。
- **虫**: クリック／タップした位置に虫を 1 体出現させ、不規則ダッシュ（Erratic）で素早く飛び回らせて、やがて画面外へ抜けると消えます。連続クリックで複数を同時に出せます（同時上限あり）。虫が居る間は羽音が鳴ります。

### オブジェクト種別

| 種別 | 表示名 | 猫用動画モードでの動き | 向きの表現 | SE |
|------|--------|------------------------|------------|----|
| `mouse` | ネズミ | 横断（Cross、上下ドリフト＋揺らぎ） | 360 度回転（rotate） | 鳴き声（チュー、断続）＋走行音（速度連動ループ） ※どちらも実録サンプル |
| `foxtail` | 猫じゃらし | 縁から出入りしつつ支点まわりに大きく揺れる（Dangle） | 水平反転なし（回転 sway が主） | なし |
| `toys` | おもちゃ | 縁から出入りしつつ揺れる（Dangle） | 水平反転なし（回転 sway が主） | なし |
| `insect` | 虫 | 不規則ダッシュ（Erratic、進入→高速ダッシュ／短い停止→急旋回→退場） | 360 度回転（rotate） | 羽音（速度連動ループ、鳴き声なし） ※Web Audio 合成 |
| `custom` | カスタム画像 | 横断（Cross） | 水平反転のみ（上下反転はしない） | なし |

- 向きの表現は 2 方式です。`rotate`（速度方向へ全方位回頭）と `flip`（進行方向で水平反転）。
- カスタム画像は猫用動画モード専用（オプションで画像を設定すると出現候補に加わる）で、マウス操作モードの操作対象には含まれません。
- SE はオブジェクト種別ごとにルーティングされ、その種別が画面上に居るときだけ鳴ります（虫だけの時にネズミの鳴き声が混ざる、といったことは起きません）。走行音・羽音は移動速度に連動して音量が上下します。

### 捕獲フィードバック

猫用動画モードで画面（キャンバス）をタップ／クリックすると、当たった中で最も近いオブジェクトが素早く逃走（一定速度で直進して画面外へ→消滅）し、反応 SE を鳴らします。鳴き声を持つ種別はその声を、持たない種別は汎用のキャッチ SE（Web Audio 合成）を再生します。同時に、当たった位置へ短命なリングの視覚バースト演出を出します。空きスペースのタップは無反応です。オプション画面を開いている間・自動停止中・一時停止中は無効です（マウス操作モードのタップは各操作対象の固有挙動に使われます）。

### 表示・操作

- **全画面表示**: オプション画面「表示」の全画面トグル、またはキーボードの `f` で全画面を切り替えます（Fullscreen API 非対応環境ではボタンを出しません）。
- **マウスカーソル非表示**: オプション（既定 OFF）で、プレイ領域（`#app`）のマウスカーソルを隠せます（猫が物理カーソルの矢印を追う誤作動を防ぐ）。設定パネルを開いている間は人間が操作できるよう通常表示に戻ります。
- **キーボードショートカット**: `Space`＝一時停止トグル / `f`＝全画面トグル / `m`＝モード切替。修飾キー付き・IME 変換中・入力欄フォーカス中・設定パネル表示中は無効化されます。
- **タッチ／タブレット最適化**: プレイ領域は `touch-action: none`・タップハイライト無効・長押し選択／コールアウト抑止で、ポインタ入力が遅延なく即発火します（`index.html`）。

### 一時停止・遊びすぎ防止

- **一時停止（合成pause）**: 「設定パネル表示中」「タブ非表示（背景タブ）」「`Space` トグル」のいずれかが真のとき現行モードを一時停止します。3 要因の論理和で管理するため、片方が解除されても他要因が残っていれば一時停止のままです。一時停止中は環境音ループ（走行音・羽音）も即座に無音化します。
- **タブ非表示で自動一時停止**: `visibilitychange` を購読し、背景タブになると自動的に一時停止して環境音ループの無駄鳴りを止めます。
- **遊びすぎ防止タイマー**: 猫用動画モードで一定時間（オプションで 5／10／15／30 分、既定＝なし）遊ぶと自動停止し、動くオブジェクトを消して無音の穏やかな背景だけにします。再開オーバーレイの「再開」を押すか、設定で上限を変えると再開します。累積は active 再生時間のみ（一時停止中は加算しません）。

### 背景・オプション・永続化

- **背景**: 単色（カラーピッカー）またはユーザー画像を選べます。画像はアスペクト比を保って画面全体を覆う cover-fit で表示されます。
- **オプション画面**: 右下の歯車ボタンから開き、中央寄せの大きめポップアップ（モーダル）で表示します。設定は **3 タブ** に整理されています。
  - **共通**: モード切替／動きの速さ（全体倍率）／表示（マウスカーソル非表示・全画面）／音量・ミュート／背景（色・画像・リセット）／オブジェクト画像（カスタム画像クリッターの設定・削除）。
  - **マウスモード**: 操作するもの（操作対象の種別）。
  - **動画モード**: 出現プリセット（賑やか／標準／控えめ）／出現間隔／出現する種類の ON/OFF／遊びすぎ防止。
  - 閉じる導線は ×ボタン／`Esc`／パネル外（バックドロップ）クリックの 3 系統です。各タブのコントロールは常時編集可（別モードの設定も事前に調整できます）。
- **出現プリセット**: 「賑やか（高密度・全種）」「標準（既定相当）」「控えめ（ゆったり間隔・虫を外す）」をワンタップで適用し、出現間隔と出現する種類をまとめて切り替えます。
- **動きの速さ**: 画面内オブジェクトの動きの速さの全体倍率（既定 1.0）。マウス操作モード・猫用動画モードの両方に効きます（出現頻度＝出現間隔とは独立）。
- **音量・ミュート**: master 音量スライダとは独立した一括ミュート（映像のみモード）を持ちます。ミュート中でも音量値は保持され、解除で元に戻ります。
- **永続化**: 設定値（背景の種類/色/画像 ID・音量・ミュート・カーソル非表示・モード・操作対象・出現間隔・遊びすぎ防止上限・カスタム画像 ID・無効化種別・動きの速さ）は `localStorage` に、画像バイナリは `IndexedDB`（背景用・クリッター用の 2 ストア）に保存します。
- **画像取り込みの堅牢化**: 受理する画像形式は `png` / `jpeg` / `webp` に限定します。デコード後の画素寸法が上限（最大辺 2048px）を超える画像は自動的に等比縮小してから使用します（VRAM の肥大を防止）。

### 効果音

効果音は **実録 WAV サンプル** と **Web Audio API による合成** を併用します。

- **ネズミの走行音・鳴き声は実録サンプル**（`public/assets/audio/`）を使います。走行音（`se_run_1..3.wav`）は 3 種を周回ごとにランダム選択してギャップレス（無音の切れ目なし）にループ連結し、移動速度に連動して音量が上下します。鳴き声（`se_squeak_1_chuchu.wav` / `se_squeak_2_single.wav` / `se_squeak_3_short.wav`）は発火のたびに 3 種からランダムに 1 発鳴らします。猫用動画モードでは断続的に、マウス操作モード（ネズミ）ではクリック／タップでも鳴ります。
- サンプルは合成 SE と同じ id で登録され、登録があれば合成より優先されます。サンプルは起動時に fetch＋decode し、失敗したグループは合成 SE にフォールバックします（壊れても無音にしない）。
- **虫の羽音・捕獲キャッチ音は Web Audio 合成**のままです。羽音は移動速度に連動するループ、キャッチ音はワンショットです。
- ブラウザの自動再生制限に対応し、最初のユーザー操作（`pointerdown` / `keydown` / `touchstart`）で `AudioContext` を resume します。master 音量・一括ミュートを持ち、context 生成に失敗した環境では全 API が安全な no-op になります。

### 起動失敗時のフォールバック

起動（bootstrap）が失敗しても画面が真っ白のままにならないよう、穏やかな日本語のフォールバックメッセージ（DOM オーバーレイ）を表示します（WebGL/WebGPU 非対応・初期化例外・`#app` 不在など、原因を問わず適用）。

## アーキテクチャ

コードは **純ロジック層** と **PixiJS 描画層** に分離しています。

- **純ロジック層**（PixiJS / DOM 非依存）: ベクトル演算、world 境界、動きの戦略、SE のパラメータ写像、設定の正規化、タブ／ショートカットのキー写像、遊びすぎタイマーなど。副作用を持たない純関数・純データとして書かれており、Vitest で単体テストしています。
- **PixiJS 描画層**: `Application` / `Container` / `Sprite` / `MeshRope` などの描画と、純ロジック層の状態（`CritterState`）を表示へ同期する処理。

主要な設計は次の通りです。

- **Critter = view + state**: 純データの `CritterState`（位置・速度・向き・回転・heading）と、それを描画へ同期する `Critter`（PixiJS）を分けています。
- **Movement（Strategy パターン）**: 動きは差し替え可能な戦略として実装します（`MouseFollowMovement` / `CrossMovement` / `DangleMovement` / `ErraticMovement` / `FleeMovement`。ほかに `DriftMovement` あり）。同じ種別をモードごとに別の動きで使い回せます。
- **CritterType（レジストリ）**: オブジェクト種別は「型定義 1 つ＋アセット」で追加できます。テクスチャ・サイズ・向きの表現方式・SE・尻尾/揺れ設定・出現計画をまとめて持ちます。
- **Mode（出現方針）**: `ManualMode` / `AutoMode` が critter・入力配線・SE の確保/解放（start/stop）と毎フレーム更新を担います。
- **ManualMode（コーディネータ + コントローラ）**: `ManualMode` は選択中の操作対象コントローラ 1 本を保持し、start/stop/一時停止/速度倍率/更新/クリックを委譲します。実挙動は `ManualController` 実装（`FollowManualController`＝カーソル追従／`FoxtailManualController`＝端から差し込む猫じゃらし／`InsectManualController`＝クリック出現の虫）が担い、種別切替では旧コントローラを破棄→新規生成して差し替えます。
- **world 座標と画面外バッファ**: critter は中心座標で管理し、画面（viewport）に各辺 margin を足した world 領域内を動けるため、画面外へ完全に隠れられます。margin は登録種別の到達距離（本体＋尻尾/揺れ）から動的に算出します。
- **SE と永続化**: 実録サンプル/合成のワンショット・ループ SE のバンク登録（同 id はサンプルが合成より優先）と種別別ルーティング、`localStorage` + `IndexedDB` による設定・画像の永続化。

## 実動作確認（QA）

本リポジトリでは、実際の描画と音の動作確認に agent-browser CLI をヘッドレスで用いる方針です。描画（スクリーンショット）に加え、音は master 出力の RMS / peak を取得して客観的に確認します。

そのための **DEV フック**（`__catScene` / `__catAudio` / `__catSettings` / `__catBg`）を、開発サーバでのみ `window` に露出しています（`import.meta.env.DEV` ガード内で定義しているため、本番ビルドでは tree-shake され残りません）。critter 数・モード・操作対象・強制 spawn・座標取得・音量/RMS・サンプル情報・ポインタ/タップの注入・遊びすぎタイマー/タブ非表示の観測などを、これらのフック経由で観測・操作できます。
